## Ureca Document Parser에 오신 것을 환영해요

Ureca Document Parser는 한국어 워드프로세서 문서(HWP/HWPX)를 Markdown으로 변환하는 파이썬 라이브러리예요. LangChain과 완벽하게 통합되어 RAG 파이프라인 구축에 최적화되어 있어요.

## 간단한 예제

```python
from ureca_document_parser import convert

# 파일로 저장
convert("보고서.hwp", "보고서.md")

# 문자열로 반환
markdown = convert("보고서.hwp")
print(markdown)

# LangChain 청크로 반환 (RAG용)
chunks = convert("보고서.hwp", chunks=True, chunk_size=1000, chunk_overlap=200)
for chunk in chunks:
    print(chunk.page_content)
```

<Note>
  **LangChain 통합 사용 시** `langchain` extra를 함께 설치해야 해요. 자세한 내용은 [설치 가이드](/installation)를 참고하세요.
</Note>

## CLI 도구

커맨드라인에서 바로 사용할 수 있는 CLI 도구도 제공해요:

```bash
# 파일 변환
ureca_document_parser 보고서.hwp -o 보고서.md

# 표준 출력으로
ureca_document_parser 보고서.hwp

# 지원 포맷 확인
ureca_document_parser --list-formats
```

## 지원 포맷

| 포맷 | 확장자 | 설명 |
|------|--------|------|
| HWP | `.hwp` | 아래한글 바이너리 포맷 (v5.0+) |
| HWPX | `.hwpx` | 아래한글 XML 포맷 (2014+) |

<Warning>
  현재 버전에서는 이미지 추출, 도형/차트, 머리글/바닥글, 각주/미주, 복잡한 표 병합 구조를 지원하지 않아요.
</Warning>
