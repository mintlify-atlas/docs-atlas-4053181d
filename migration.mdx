## 개요

Ureca Document Parser는 현재 활발히 개발 중인 알파 단계(v0.0.x)예요. 이 페이지에서는 버전 간 주요 변경사항과 마이그레이션 방법을 안내해요.

<Warning>
**알파 버전 주의사항**: v1.0 릴리스 전까지는 마이너 버전(0.x.0)에서도 breaking changes가 발생할 수 있어요. 프로덕션 환경에서는 버전을 고정(`==0.0.9`)해서 사용하는 것을 권장해요.
</Warning>

## 현재 버전

**최신 버전**: v0.0.9 (2024)

```bash
pip install ureca_document_parser==0.0.9
```

## 버전 업그레이드 체크리스트

새 버전으로 업그레이드하기 전에 다음 사항을 확인하세요:

<Steps>
  <Step title="의존성 확인">
    Python 버전 요구사항을 확인하세요. 현재 Python 3.12+ 필수예요.
    
    ```bash
    python --version  # 3.12 이상 확인
    ```
  </Step>
  
  <Step title="변경 로그 읽기">
    [GitHub Releases](https://github.com/ureca-corp/document_parser/releases)에서 변경사항을 확인하세요.
  </Step>
  
  <Step title="테스트 환경에서 먼저 검증">
    프로덕션 적용 전 개발/스테이징 환경에서 충분히 테스트하세요.
    
    ```bash
    # 가상 환경에서 테스트
    uv venv test-env
    source test-env/bin/activate
    uv pip install ureca_document_parser==<new-version>
    # 기존 코드 실행 및 테스트
    ```
  </Step>
  
  <Step title="업그레이드 실행">
    문제가 없으면 프로덕션 환경에 적용하세요.
    
    ```bash
    pip install --upgrade ureca_document_parser
    ```
  </Step>
</Steps>

## 알려진 문제 및 해결 방법

### Python 3.11 이하 호환성

**문제**: Python 3.11 이하에서는 설치가 불가능해요.

```
ERROR: Package requires Python >=3.12
```

**해결 방법**: Python 3.12 이상으로 업그레이드하세요.

```bash
# pyenv 사용 시
pyenv install 3.12.0
pyenv global 3.12.0

# 또는 시스템 패키지 관리자로 설치
sudo apt install python3.12  # Ubuntu/Debian
brew install python@3.12     # macOS
```

### LangChain 버전 충돌

**문제**: 기존 프로젝트의 LangChain 버전과 충돌할 수 있어요.

```
ERROR: Cannot install langchain-core==0.1.0 and langchain-core>=0.2
```

**해결 방법**: `langchain` extra 없이 설치하고, LangChain 통합을 직접 구현하세요.

```bash
# langchain extra 제외하고 설치
pip install ureca_document_parser

# 수동으로 청킹 구현
from ureca_document_parser import convert
from langchain.text_splitter import RecursiveCharacterTextSplitter

markdown = convert("document.hwp")
splitter = RecursiveCharacterTextSplitter(chunk_size=1000)
chunks = splitter.create_documents([markdown])
```

### 파일 인코딩 문제

**문제**: 윈도우 환경에서 한글 파일명이 깨지는 경우가 있어요.

**해결 방법**: 파일 경로를 명시적으로 UTF-8 인코딩하세요.

```python
from pathlib import Path

file_path = Path("문서.hwp")  # Path 객체 사용 권장
convert(file_path, "output.md")
```
