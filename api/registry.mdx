## 개요

`FormatRegistry`는 파일 확장자를 파서에 매핑하고, 포맷 이름을 Writer에 매핑하는 중앙 관리 시스템이예요. 전역 싱글톤으로 동작하며 스레드 안전하게 구현되어 있어요.

## FormatRegistry 클래스

### 클래스 시그니처

```python
class FormatRegistry:
    """Central registry for parsers and writers.
    
    Parsers are registered by file extension (e.g. '.hwp').
    Writers are registered by format name (e.g. 'markdown').
    """
```

### 생성자

```python
def __init__(self) -> None
```

새로운 `FormatRegistry` 인스턴스를 생성해요. 일반적으로 직접 생성하지 않고 [get_registry()](#get_registry)를 통해 접근해요.

### 메서드

#### register_parser()

<ParamField path="cls" type="type[Parser]" required>
  등록할 파서 클래스예요. [Parser 프로토콜](/api/models#parser-protocol)을 만족해야 해요.
</ParamField>

```python
def register_parser(self, cls: type[Parser]) -> None
```

파서 클래스를 등록해요. 파서의 `extensions()` 메서드가 반환하는 모든 확장자에 대해 매핑이 생성돼요.

**예시:**
```python
from ureca_document_parser import get_registry
from my_parser import CustomParser

registry = get_registry()
registry.register_parser(CustomParser)
```

#### register_writer()

<ParamField path="cls" type="type[Writer]" required>
  등록할 Writer 클래스예요. [Writer 프로토콜](/api/models#writer-protocol)을 만족해야 해요.
</ParamField>

```python
def register_writer(self, cls: type[Writer]) -> None
```

Writer 클래스를 등록해요. Writer의 `format_name()` 메서드가 반환하는 포맷 이름으로 매핑돼요.

**예시:**
```python
from ureca_document_parser import get_registry
from my_writer import HTMLWriter

registry = get_registry()
registry.register_writer(HTMLWriter)
```

#### parse()

<ParamField path="filepath" type="Path | str" required>
  파싱할 파일 경로예요.
</ParamField>

```python
def parse(self, filepath: Path | str) -> Document
```

파일 확장자에 맞는 파서를 자동으로 선택하여 파일을 파싱해요.

<ResponseField name="return" type="Document">
  파싱된 [Document](/api/models#document) 객체예요.
</ResponseField>

**예외:**
- `ValueError`: 지원하지 않는 파일 확장자일 때 발생해요. 오류 메시지에 지원되는 확장자 목록이 포함돼요.

**예시:**
```python
from ureca_document_parser import get_registry

registry = get_registry()
doc = registry.parse("보고서.hwp")
print(f"제목: {doc.metadata.title}")
```

#### write()

<ParamField path="doc" type="Document" required>
  변환할 Document 객체예요.
</ParamField>

<ParamField path="format_name" type="str" required>
  출력 포맷 이름이예요 (예: `"markdown"`).
</ParamField>

```python
def write(self, doc: Document, format_name: str) -> str
```

지정한 포맷의 Writer를 사용하여 Document를 문자열로 변환해요.

<ResponseField name="return" type="str">
  변환된 출력 문자열이예요.
</ResponseField>

**예외:**
- `ValueError`: 지원하지 않는 출력 포맷일 때 발생해요. 오류 메시지에 지원되는 포맷 목록이 포함돼요.

**예시:**
```python
from ureca_document_parser import get_registry

registry = get_registry()
doc = registry.parse("보고서.hwp")
markdown = registry.write(doc, "markdown")
```

## get_registry()

### 자동 등록되는 컴포넌트

`get_registry()`를 처음 호출하면 다음 컴포넌트들이 자동으로 등록돼요:

**파서:**
- `HwpParser` - `.hwp` 파일 지원
- `HwpxParser` - `.hwpx` 파일 지원
- `PdfParser` - `.pdf` 파일 지원 (선택적, `pymupdf` 설치 필요)

**Writer:**
- `MarkdownWriter` - `markdown` 포맷 지원


```python 지원 포맷 확인

registry = get_registry()

print("입력 포맷:")
for ext in registry.supported_extensions:
    print(f"  - {ext}")

print("\n출력 포맷:")
for fmt in registry.supported_formats:
    print(f"  - {fmt}")
```

```python 커스텀 파서 등록
from pathlib import Path
from ureca_document_parser import get_registry
from ureca_document_parser.models import Document, Paragraph, Metadata
from ureca_document_parser.protocols import Parser

class TxtParser:
    """간단한 텍스트 파일 파서"""
    
    @staticmethod
    def extensions() -> list[str]:
        return [".txt"]
    
    @staticmethod
    def parse(filepath: Path | str) -> Document:
        path = Path(filepath)
        text = path.read_text(encoding="utf-8")
        
        paragraphs = [
            Paragraph(text=line)
            for line in text.split("\n")
            if line.strip()
        ]
        
        return Document(
            elements=paragraphs,
            metadata=Metadata(
                source_format="txt",
                title=path.stem
            )
        )

# 커스텀 파서 등록
registry = get_registry()
registry.register_parser(TxtParser)

# 이제 .txt 파일도 변환 가능
from ureca_document_parser import convert
convert("메모.txt", "메모.md")
```

```python 커스텀 Writer 등록
from ureca_document_parser import get_registry
from ureca_document_parser.models import Document, Paragraph
from ureca_document_parser.protocols import Writer

class HTMLWriter:
    """간단한 HTML Writer"""
    
    @staticmethod
    def format_name() -> str:
        return "html"
    
    @staticmethod
    def file_extension() -> str:
        return ".html"
    
    @staticmethod
    def write(doc: Document) -> str:
        lines = ["<!DOCTYPE html>", "<html>", "<body>"]
        
        for elem in doc.elements:
            if isinstance(elem, Paragraph):
                if elem.heading_level:
                    tag = f"h{elem.heading_level}"
                    lines.append(f"<{tag}>{elem.text}</{tag}>")
                else:
                    lines.append(f"<p>{elem.text}</p>")
        
        lines.extend(["</body>", "</html>"])
        return "\n".join(lines)

# 커스텀 Writer 등록
registry = get_registry()
registry.register_writer(HTMLWriter)

# HTML로 변환
from ureca_document_parser import convert
from pathlib import Path

doc = registry.parse("문서.hwp")
html = registry.write(doc, "html")
Path("output.html").write_text(html, encoding="utf-8")
```

```python 에러 처리
from ureca_document_parser import get_registry, ParseError

registry = get_registry()

try:
    # 지원하지 않는 확장자
    doc = registry.parse("문서.docx")
except ValueError as e:
    print(f"입력 포맷 오류: {e}")
    print(f"지원 확장자: {registry.supported_extensions}")

try:
    # 지원하지 않는 출력 포맷
    doc = registry.parse("문서.hwp")
    output = registry.write(doc, "json")
except ValueError as e:
    print(f"출력 포맷 오류: {e}")
    print(f"지원 포맷: {registry.supported_formats}")
```

## 내부 구현

### 스레드 안전성

레지스트리는 더블 체크 락킹(Double-Checked Locking) 패턴으로 구현되어 멀티스레드 환경에서 안전해요:

```python
_registry: FormatRegistry | None = None
_registry_lock = threading.Lock()

def get_registry() -> FormatRegistry:
    global _registry
    if _registry is not None:
        return _registry
    with _registry_lock:
        if _registry is None:
            registry = FormatRegistry()
            _auto_register(registry)
            _registry = registry
    return _registry
```

### 자동 등록 과정

`_auto_register()` 함수가 내장 컴포넌트를 자동으로 등록해요:

1. **필수 컴포넌트**: 항상 등록됨
   - HwpParser, HwpxParser
   - MarkdownWriter

2. **선택적 컴포넌트**: 의존성이 있을 때만 등록됨
   - PdfParser (pymupdf 필요)

## 참고사항

<Note>
일반적인 사용 시나리오에서는 레지스트리에 직접 접근할 필요가 없어요. [convert()](/api/convert) 함수가 내부적으로 레지스트리를 사용해요.
</Note>

<Tip>
새로운 포맷을 추가하려면 [포맷 확장 가이드](/extending)를 참고하세요. Parser 또는 Writer 프로토콜을 구현하고 레지스트리에 등록하면 돼요.
</Tip>

<Warning>
레지스트리는 싱글톤이므로, 한 번 등록된 파서나 Writer는 애플리케이션 전체에서 공유돼요. 테스트 환경에서는 주의가 필요해요.
</Warning>
